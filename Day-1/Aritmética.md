# Aritmética
É possivel realizar calculos matematicos dentro função `console.log()`.

## Exemplo
```js
console.log(2 + 3); // adição
console.log(3 - 2); // subtração
console.log(2 * 3); // multiplicação
console.log(3 / 2); // Divisão
console.log(3 % 2); // Modulo - encontrar o restante da divisão
console.log(3 ** 2) // Exponenciação 3 ** 2 == 3 * 3
```
